# Magic Prompt

**Локальный помощник для доработки промптов**  
Превратит идею → в чёткий промпт для ChatGPT, MidJourney и др.


---

## Что делает?
- Уточняет и детализирует промпты
- Работает **офлайн** через `ollama` + `mistral`
- Быстро, приватно, без API

---

## Запуск

Пример
text> Сделай промпт для логотипа кофейни
→ Получишь готовый промпт с цветами, стилем, форматом.

Дальше

Сохранение промптов
Шаблоны (код/дизайн/текст)
Веб-интерфейс

# 1. Установи Ollama и модель

# 3. Запусти
